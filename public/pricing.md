# SafeJSON Pricing

**Last updated:** 2026-06-16
**Website:** https://www.safejson.dev

## Free

- **Price:** $0/month
- **Features:** JSON Formatter, JSON Validator, JSON Beautifier, JSON Viewer, JSON Parser, CSV to JSON, JSON to CSV
- **Privacy boundary:** no pasted-content upload for core formatting workflows

## Pro

- **Price:** $5/month or $39/year
- **Device activations:** up to 2 device activations per license
- **Workflow positioning:** SafeJSON Pro is for sensitive JSON workflows
- **Features:**
  - JSON Diff for API/webhook/config payloads
  - JWT claim inspection
  - JSONPath query for nested API responses
  - Schema validation for API/webhook payloads
  - 5 free runs before upgrading

## Team Lite Demand Test

SafeJSON is testing interest in a small dev team license for teams handling API payloads, JWTs, schemas, webhook payloads, and configs.

- Simple team license
- Invoice-friendly purchase can be discussed
- Small team licenses can start around 5 seats
- No SSO, admin dashboard, compliance package, or SLA
- Request path: https://www.safejson.dev/pricing

## Payment

- **Processor:** Lemon Squeezy
- **Billing:** Monthly or annual (auto-renewing, cancel anytime)
- **License delivery:** License key sent by email after checkout
- **Activation:** Paste license key on https://www.safejson.dev/unlock

## Privacy

- Core tool operations use a browser-local workflow with no pasted-content upload
- SafeJSON uses Google Analytics for aggregate product analytics
- SafeJSON does not send pasted JSON, JWT tokens, schemas, queries, or license keys to analytics
- Verify: open DevTools Network tab while using any tool
