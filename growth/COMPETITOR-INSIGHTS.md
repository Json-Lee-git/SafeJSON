## JSON Crack / ToDiagram

### Relationship to SafeJSON

JSON Crack is not a direct competitor — it is an adjacent tool / mirror. It
proves that a path exists: open-source JSON tool → GitHub distribution →
multi-platform distribution → commercial conversion.

### What JSON Crack does well

- **JSON visualization.** Interactive node-based graphs. Screenshots are
  inherently shareable — the core driver of 44K+ GitHub stars.
- **GitHub stars.** Accumulated over 4 years (2022–2026) through Show HN (576
  points), visual virality, and consistent maintenance.
- **VS Code distribution.** 728K+ installs via the VS Code Marketplace — far
  more reach than SEO alone.
- **Open-source trust.** GPL v3 license. Self-hostable. Users can fork and run
  locally.

### What JSON Crack exposes about the market

- Open-source JSON tools have genuine propagation potential.
- The "free tool → commercial product" model works (ToDiagram, ~$9/mo per G2).
- IDE/extension distribution is a stronger growth lever than pure SEO for
  developer tools.
- Visual differentiation drives organic sharing.

### What SafeJSON should learn

- Visual proof assets (DevTools verification GIF, JWT no-upload screenshot)
  are the equivalent of JSON Crack's node graphs — the shareable visual.
- GitHub README quality directly drives stars and trust.
- Browser extension distribution (Edge + Chrome) plus future VS Code extension
  is the right long-term play.
- Free core + paid advanced workflow is validated by JSON Crack/ToDiagram.

### What SafeJSON should NOT copy

- **Graph visualization direction.** Different product, different strengths.
- **Split-brand monetization.** JSON Crack → ToDiagram creates brand
  confusion. SafeJSON is a single brand with a direct upgrade path.
- **Chasing accumulated stars.** 4 years of 44K stars is not the goal.
  Consistent content + distribution is.
